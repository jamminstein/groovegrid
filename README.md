# groovegrid

soulful grid bass / disco funk chicago house for monome norns + grid

## features

- 4-voice poly synth (Get Lucky, Da Funk, Around World, Harder Better)
- 8 arp modes (up, down, up-down, random, drunk, skip, converge, stutter)
- ripple visual fx on grid
- 12 chord progressions (disco ghost, deep house, funk glitch, etc.)
- 16 funky scales (Aeolian, Dorian, Pentatonic, Blues)
- wah effect with rate/depth control
- pulsing metronome row
- lenny face grid animations
- MIDI out

## controls

- **K1**: toggle chord progressions
- **K2**: randomize scale + layout
- **K3**: toggle arpeggiator
- **E1**: cycle progressions (prog) / BPM (arp) / octave
- **E2**: prog division / arp mode / volume
- **E3**: wah depth / envelope shape

## grid layout

- **row 1**: voice select (4 voices x 4 cols each)
- **rows 2-7**: play pads (scale-mapped notes)
- **row 8**: pulsing metronome

## requirements

- monome norns
- monome grid
- GrooveGrid SuperCollider engine
